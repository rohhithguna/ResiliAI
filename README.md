# AI SRE Autonomous Recovery System

## Overview
An AI system for automated incident recovery using a multi-agent architecture.

## Architecture
- Frontend Agent
- Backend Agent
- Database Agent
- Traffic Agent
- Queue Agent
- Coordinator Agent

## Key Features
- OpenEnv compliant environment
- Deterministic evaluation (tasks + graders)
- Multi-agent decision system
- Robust handling of failures, latency, and traffic
